更新About和Work

更新About直接编辑 Hexo/ 目录下的index.html，然后将public中的index.html替换

更新Work直接编辑 Hexo/public/work/ 目录下的index.html

更新Photography

在 photography/insPicture/ 目录中添加图片文件，然后编辑photography/ 目录下的index.html

构建和部署

构建


hexo g

部署


hexo d

需要注意的是

  1. 构建命令执行之后，Hexo/public/ 目录下的index.html文件夹会被脚本文件替换为默认样式，因此需要使用 Hexo/ 目录下的 index.html 来手动替换
  2. 构建命令执行之后，Hexo/public/Archives/ 目录下的 index.html 文件导航栏会被脚本替换为默认样式，因此需要手动注销其中的 “DateWith” 模块。
