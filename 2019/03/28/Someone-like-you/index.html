Someone like you

inside SteveChao
2019-03-28

难得寝室只有我一个人，阳台敞开着，坐在桌前的椅子上，午睡后的疲倦和轻松还没有完全散去。长沙开年以来第一次吹这么舒服的风，像是极其精确地在提醒着春天来了。多日来的烈烈的风和冷雨让长沙的三月和春暖花开没有丝毫联系。此刻耳机里单曲循环着”someone like you”钢琴改编版，很舒缓轻松，没有Adele的原作中的回忆不舍和惋惜。只剩下祝福，像是两个老朋友坐在草地上你一句我一句地聊天。

刚才梦到了胡筱岩带着高中的全班春游来着，去的依然是中梁山，所以是关于高二那年的春游远足的记忆。那个春天我喜欢上了一个对后来的我影响很大的姑娘。梦里的春游只有两个确切的人：胡筱岩和戴诚。对戴诚的记忆从高一认识开始，到高二就突然没有了。梦里戴诚和我在队伍最前面，应该是对早操的记忆，我拿了那杆西红柿鸡蛋旗将近三年，关于旗有很多很不错的故事和回忆。

这个春天有一个陪了我很久的人离开了我，我曾深爱她。

四季里最喜欢春。尤其是天水的春天，长沙是少有春的。离院子二十米有片小小的杏园，属于姥姥的姨，老人已不怎么管了，便从小便宜了我。三月正是杏花开的时候，杏树不高很好爬。杏花开总是仿佛在一瞬间，像是前一秒还是从冬天走出的黑漆漆光秃秃的木头，后一秒就被白色和小花和蜜蜂包裹了。对小时候的记忆几乎没有学校，全是在家和在野外疯跑。杏树上总是分泌树胶，粘在衣服上会变黑色一块就像是泡泡糖一样，与泡泡糖不同的是在水里一泡就掉了。我几乎有段时间会长在杏树上，最常爬上去的那棵和其上有其总是承受着我的那一支树干被我的衣服磨的光滑，算是被我盘出来的。杏花开时树丛中的蜜蜂多到像是跨年夜的步行街。但我却从来没被蛰过。也不总是在树上，曾经在两棵树中间绑了个吊床，中午十一点躺在上面，阳关被小花割碎从缝隙中洒下来掉在脸上，三月的微风吹过会有花瓣飘落，蜜蜂会从面前快速飞过，是频率很高的一声“嗡”，而数不清的满树的蜜蜂在一起就成了一种厚重而有些闷的“嗡嗡”的声音，我就像是躺在一个由清清的花香和嗡嗡盖出来的帐篷中一样。

自家院子外还有一棵樱桃树，樱桃的花期和杏差不多，与矮大多支的黑漆漆杏树不同的是，樱桃树总是长的很好看，主干直而且光滑，树枝也是很有层次的一层一层平平地长出来。樱桃树的颜色是偏褐色的紫色。樱桃花要比杏花大一些，像梅花大小。花蕊也要茂盛很多。天水是没有樱花的，小时候我曾一度以为樱花就是樱桃的花。我喜欢周末的上午捧一本书去樱桃树下坐着，那是很难有的轻松和平静。

靠，一瞬间寝室又全是人了，不写了。
