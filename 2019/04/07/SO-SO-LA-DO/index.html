SO SO LA DO

inside SteveChao
2019-04-07

“我不过像你像他像那野草野花，冥冥中这是我唯一要走的路啊。”

陪老姐在长沙逛了三天，很久没有这么累过了。这一个多月来，多数时候的生活是床下吃外卖，床上翻手机，有意义的产出少的可怜可忽略。似乎是找不到什么动力了。长沙的天气突然变热，从三天前的捂着棉袄颤抖到今天中午背心短裤还得擦汗，而明天则是不出意外的又一次降温至十度，像极了过山车、也像是多数感情：冷静的初识 —— 火热的爱恋 —— 回归冰冷的最后一句再见。

如同一个连着发了三天脾气终于停下来去吸支烟的巨人一样，晚上七点，热浪有了些许收敛。宿舍门同阳台落地窗此刻敞开着，不时有穿堂风吹过，让人突然精神。四年前的夏天晚上也是这样，在天水一中高三楼中灯火通明的自习室中。随着入夜降温而下沉的山间空气顺着牛家山中的条条沟壑流向山下的教学楼，穿过伏金祥校长引以为高级的向上开的推拉窗吹起我的书角，也吹起她垂下的鬓角，带着桃花的清香和微弱的虫鸣。地上总是湿的，炎热的夏日没有空调的北方教室都是这样降温。

坐在后面的胡祎像个假冒伪略的自以为是的哲学家，总是喜欢跟我讲一些莫名其妙的大道理，模糊厚重的两片椭圆凹透镜让他的眼睛小了又小，但掩盖不住他的自信，他总是那么自信。不自信的人是不敢在全班师生面前将 “他是二号人物” 继往开来地翻译为 “He sits in the second chair”(他坐第二把交椅) 的。他也是我所见过的喝中药最多的男孩子，高三的孩子都是需要补营养的，排骨汤、维他命、水果沙拉、热牛奶。。。而胡祎则是每晚一袋的中药汤，会让儿误以为是提高智力的某种神秘偏方，事实是这位可怜的哲学家被感冒折磨了快半年了。

王建是贯穿我高中三年的男生，他是我有幸认识的不多聪明人中特别机灵的一个，但好像是进入高三之后他就突然不会考试了。高三的他被成绩折磨，骑车穿过半个小城上晚自习不够，还要去上补习班，好在最后高考发挥蛮好，去了一很棒，而且校园特漂亮的学校。于我眼里，高中的他就是个“万人迷”，任何人都喜欢和他说学逗唱。和王建聊天是总能感受到他轻快的幽默和充沛的活力，像是一只三个月大的狗。我最喜欢听他用天水话讲老高，他的初中数学老师。我没见过老高，在我想象中老高是一个寸头、有点啤酒肚，花白格子短袖扎在没有拉拉链的裤子的裤腰中的三十多岁的男人。王建以后的孩子绝对会是同龄人中天水话讲的最地道的，因为王建对天水话的钟爱是我见过最甚的，他用天水话跟我讲老高的段缎趣事、用天水话跟我解释“驴姐”称号的诞生、用天水话让我晚自习后送他回家以防强人出没。。。前一段时间他生日，问他接下来怎么规划，他说要一边工作一边复习准备二战，替他感到辛苦，希望他能顺利上岸。

本来意犹未尽还想写点什么，但突然随风吹来的一丝寒意带来一些莫名的失落和伤感，那就这样吧。
